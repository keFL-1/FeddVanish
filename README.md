# FeddVanish
FeddVanish — легкий плагин для Paper 1.20.6. 
✅ Полная невидимость игрока (hidePlayer). 
✅ Скрытие из TAB  
✅ Мобы игнорируют игрока. 
✅ Настраиваемый Actionbar и сообщения. 
✅ Отключение подбора предметов в ванише.  
Команды: /vanish, /v. 
Права: feddvanish.use. 
Идеально для скрытого модерирования без режима Spectator.
